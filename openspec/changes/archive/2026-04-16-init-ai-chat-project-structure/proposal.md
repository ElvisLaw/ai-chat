## Why

我们需要为 AI Chat 项目建立一个清晰的基础结构，包括标准的目录布局和依赖声明。这是后续所有开发的基础，没有它项目会缺乏组织性。

## 什么会改变

- 创建标准的 Python 项目目录结构（src/、tests/、config/ 等）
- 添加 `pyproject.toml` 声明项目元数据和依赖
- 创建虚拟环境配置
- 添加 `.gitignore`、`README.md` 等配置文件
- 列出并记录 AI Chat 相关的 Python 包

## 新能力

- **项目结构初始化**：标准的 Python 项目布局
- **依赖声明**：通过 pyproject.toml 集中管理所有项目依赖

## 影响

- 创建约 5-8 个新文件
- 不影响现有代码（目前项目为空）
